# Agent Usage History — shared-instructions

This file records agent usage specific to the shared-instructions repository (meta-work: instructions, docs, scripts, config).

Standard entry format:
- `[YYYY-MM-DD HH:MM] agent=<name> task=<type> model=<model> status=<primary|fallback-x> lang=<language> desc=<short description>`

Quick logging command (from shared-instructions root):
- `./scripts/log-agent-usage.sh --agent "Custom_Auto" --task code --model claude-sonnet-4.5 --status primary --lang typescript --desc "Short action description"`

See also:
- `./scripts/stats-agent-usage.sh .agent-usage.md` — Box-style stats
- `./scripts/stats-agent-usage-pie.sh .agent-usage.md` — ASCII pie view
